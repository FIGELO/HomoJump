# HomoJump
一款利用 HarmonyOS 无障碍能力 [Accessbility Kit](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/accessibilitykit-V5) 帮助用户使用手机的应用。
可以通过导入配置规则，实现自定义的屏幕点击功能

## 功能与实现进度

### 目前已实现

1. 获取屏幕组件树
2. 根据特定的组件信息获取组件
3. 对组件进行操作
